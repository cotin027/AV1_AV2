# Portal Geek - Geek Universe 2026

## Sobre o projeto

Esse projeto é um site feito para um evento fictício chamado Geek Universe 2026.  
Ele foi criado usando HTML, CSS e JavaScript, com o objetivo de praticar desenvolvimento web.

---

## O que o site tem

- Página inicial com informações do evento e contador de inscritos
- Página de atrações com botões de "ver mais"
- Página de cronograma com horários dos eventos
- Página de pré-cadastro com formulário

---

## Funcionalidades em JavaScript

- Botão para mostrar/ocultar informações das atrações
- Formulário com validação simples (campos obrigatórios)
- Mensagem de sucesso ao enviar o formulário
- Contador de inscritos que aumenta ao clicar em “Participar”
- Botão para alternar entre tema claro e escuro

---

## Tecnologias usadas

- HTML
- CSS
- JavaScript
- Font Awesome
- Google Fonts

---

## Como usar

É só abrir o arquivo `index.html` no navegador e navegar pelas páginas.

---

## Observação

O projeto não usa banco de dados, tudo funciona apenas no navegador.
